👥 Keen Keeper App

A responsive web app to manage friends and track interactions like calls, texts, and video chats. It helps users stay connected by monitoring communication and setting contact goals.

🛠️ Technologies Used

React.js
React Router DOM
Tailwind CSS + DaisyUI
Recharts
React Toastify
React Icons
React Helmet
React Spinner

✨ Key Features

👫 View and manage friend profiles with detailed information
⚡ Log interactions (Call, Text, Video) with toast notifications
📜 Timeline with interaction history and analytics chart
